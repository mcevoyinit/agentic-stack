## Key Metrics to Track

### Competitor Metrics
- Funding (amount, valuation, investors)
- User reach / DAU / MAU
- Revenue estimates
- CTR claims
- Team size / key hires

### Market Metrics
- TAM / SAM / SOM
- Growth rates (CAGR)
- Category spend
- Platform adoption

### Regulatory
- EU rulings (Google, Meta)
- IAB framework updates
- GDPR / CCPA changes
